# 11+ Galaxy — Future Features Backlog

> Features deferred from MVP. Reference this document when building subsequent issues.

---

## Phase 2: Enhanced Assessments

### Adaptive Difficulty Engine
- Questions get harder/easier based on child's performance
- Track per-question-type accuracy rates
- Bayesian difficulty estimation per child per topic
- Suggested difficulty curve per subject

### Extended Question Types
- Drag and drop (ordering sequences, matching pairs)
- Fill in the blank (spelling, grammar, maths)
- Timed speed rounds (mental arithmetic, quick-fire vocab)
- Cloze passages (comprehension with blanks)
- Pattern completion (non-verbal reasoning with interactive grids)
- Creative writing prompts with word count targets

### Assessment Packs & Mock Exams
- Full-length mock 11+ papers (timed, multi-section)
- Pack-based assessments (10 questions grouped thematically)
- Weekly challenge papers
- School-specific paper styles (CEM, GL, ISEB)

## Phase 4: Advanced Analytics & Personalisation

### Deep Insights
- Per-question-type accuracy trends (e.g., "struggles with long division but strong on fractions")
- Predicted score ranges for 11+ exam
- Comparison to anonymised cohort (percentile)
- Time-per-question analytics (identifies rushing or overthinking)
- Mistake pattern analysis (common wrong answer types)

### Personalised Recommendations
- AI-generated study plans based on weak areas
- Recommended next mission based on gaps
- "You should focus on..." weekly email to parents
- Smart revision scheduling (spaced repetition)

### Learning Path
- Structured curriculum paths per target school type
- Pre-assessment placement test
- Milestone checkpoints with progress gates
- Term-based planning (aligned with school year)
